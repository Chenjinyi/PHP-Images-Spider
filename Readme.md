## 使用方法

```shell
#克隆库
git clone git@github.com:Chenjinyi/PHP-Images-Spider.git
#进入文件夹
cd PHP-Images-Spider
#运行爬虫（需有PHP环境）
php spider.php
#按命令行提示输入相应的数字
```
